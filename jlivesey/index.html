What My Blog Can Do 26/03/2018

So, I'm not using anything fancy like WordPress (but I do for my main blog). I'm using HTML, CSS and JavaScript. As pure as it can get.

One of the most important things that I needed is the ability to add <code> snippets. They're great in the middle of sentences, although I don't suggest speaking code in the middle of a normal day-to-day sentence.

That's why I have <pre> code blocks at hand. And, with a copy button so you can steal use/modify/delete/do-what-ever-you-want my code. Here's an example of this:

document.onload = function() {
    alert("This thing is really cool, it can do so many great functions.");
}

Now try the copy button on this next code snippet, and paste it into the JavaScript console:

while (true) {
    alert("Hey! I'm here as a sole purpose to annoy you.");
}
